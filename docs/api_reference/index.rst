ChemBERTa4 API Reference
========================

Minimal library for molecular property prediction with OLMo-7B.
Supports classification, regression, causal LM pretraining, and instruction tuning
with QLoRA, LoRA, and full fine-tuning strategies over distributed (DDP) training.
